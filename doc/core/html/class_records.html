void Records::newRecord ( const std::string &  name,

newRecord allows to register a new score in the score panel. If the player already exists, a check of the already existing score will be done to see if the last referenced score is better. If it is the case, the user's score will be replaced. If another person has already indicated an identical score in the scoreboard, this one will be replaced in order to create a dynamic competition between the different players.
