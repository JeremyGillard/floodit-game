Detailed Description

The QFloodIt class is a representation of the teasing game adapted to the signal/slots the qt features. This class implements only a few more things than the model and simply inherits most of these methods and attributes. This allows to take advantage of the observer/observable pattern offered directly by the qt mechanisms.
